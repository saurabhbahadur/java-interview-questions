<h1 align="center" > Queue </h1>
<p align="center" > </p>


+ A queue is a linear data structure that follows the `FIFO` (First In, First Out) principle, meaning that elements are inserted from one end (the rear) and removed from the other end (the front). This makes queues ideal for scenarios like task scheduling, order processing, and breadth-first search `(BFS)` in graphs.

### Key Operation 

+ `add(element)` - Adds an element to the rear of the queue; throws an exception if the operation fails.
+ `offer(element)` - Similar to add(), but returns false if the queue cannot accommodate the new element.
+ `remove()` - Removes and returns the head of the queue; throws NoSuchElementException if the queue is empty.
+ `poll()` - Removes and returns the head of the queue, or returns null if the queue is empty.
+ `element()` - Retrieves the head without removing it; throws NoSuchElementException if the queue is empty.
+ `peek()` - Retrieves the head without removing it, or returns null if the queue is empty.
